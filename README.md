# notebooklm-mcp-main
NotebookLM MCP (Model Context Protocol) es una integración de código abierto que actúa como un puente entre NBLM y los agentes de IA compatibles con MCP, como Claude Code o Claude Desktop. Esta herramienta permite a los usuarios automatizar la interacción con NBLM, controlando cuadernos y fuentes directamente desde la línea de comandos o chat de IA
